## Para executar o projeto
Suba o dump do banco de dados  
Atualize o DATABASE_URL do .env com a configuração atual  
Execute:  
- npm install 
- npm run dev
 
Caso ocorra problemas com a biblioteca de datas com o yarn, instale-a manualmente:  
- yarn add date-fns
